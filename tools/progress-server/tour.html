PASSO 3 DI 5
Tutti i tuoi device sono uno solo
SOLEM unisce PC, laptop, Raspberry, smartphone, smart glass in un cluster privato via mesh VPN. GAVIO dispatch ogni richiesta al device più adatto.
Esempio reale
• Chiedi una traduzione veloce → laptop
• Analizza una foto → Jetson (ha la GPU)
• Cerchi "che ore sono" sul glass → risposta locale, niente roundtrip
• Inferenza LLM grande → server-nvidia
Per aggiungere un device: solem pair → ottieni un PIN → inserisci sull'altro device.
