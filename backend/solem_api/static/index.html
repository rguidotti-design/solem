    Personalizzazione
    
        
    
          SOLEM è dichiarativo: ogni preferenza vive in un file Nix versionabile.
          Modifica nixos/configuration.nix, esegui sudo nixos-rebuild switch --flake .,
          il sistema si riconfigura in modo atomico. Se qualcosa va storto:
          sudo nixos-rebuild --rollback switch torna alla generazione precedente.
        
    
        
    Cose che puoi personalizzare ora:
    
        
      
          
    • Profilo (minimal/developer/creator/server/desktop)

    • Linguaggi installati (Python/Go/Rust/Node/Zig/Deno/Lua/Ruby/Java)

    • Moduli AI (Jupyter, ML libs, modelli Ollama)
      • 
          
    • Mesh WireGuard, Zero-Trust mTLS, AppArmor selettivo

    • Desktop ambiente (Hyprland, Pipewire, Bluetooth)
