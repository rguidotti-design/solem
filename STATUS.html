Limiti e problemi NOTI 6 voci

⚠ VM TCG lentissima
KVM non disponibile in WSL2 → tutto software emulation. Soluzione: bare-metal Beelink Step 1, oppure Linux nativo con KVM
⚠ Niente GUI grafica al boot
Vedi solo console testuale + dashboard separata via browser host. Da fixare: autoLogin Hyprland (rebuild richiesto)
⚠ Build VM occasionalmente fallisce
Per problemi rete cache.nixos.org transitori o flake8 strict su Python writer. Fixare: retry automatico + flake8 ignore esteso (fatto)
⚠ Modelli Ollama on-demand
Specialist AI (Coder/Researcher/Writer) scaricano modelli al primo invoke (4-8GB cad). Pre-pull manuale necessario
⚠ Plymouth "solar" generico
Tema sole built-in NixOS, NON tema custom SOLEM con logo. Custom rimandato Step 2 (richiede asset PNG renderizzati)
⚠ Distanza enorme da "Windows/iOS feel"
SOLEM oggi è OS server-like, non desktop polishato. Stima realistica: 12-18 mesi per arrivare a UX competitiva
