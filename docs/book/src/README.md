SOLEM è il **Friday/JARVIS** di Iron Man:

- Auto-improvement loop: red-team notturno + self-heal
- Friday-style CLI unificato

## Filosofia

- **100% FOSS** — nessun servizio proprietario nei default
- **Zero-trust** — anche per la AI che ospiti
- **0 € licenze** — nessun cloud paid
- **Self-hosting** — i tuoi dati restano tuoi
- **Reproducible** — Nix garantisce build identiche
